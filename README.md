# Solargraphy Image Analysis Tool

Automated analysis of solargraphy images to extract solar trajectories, altitude profiles, and brightness characteristics.

## Features

- **Automatic Track Detection**: Identifies sun tracks using contour analysis and morphological operations
- **Altitude Profiling**: Calculates solar altitude angle and trajectory over image progression
- **Brightness Analysis**: Extracts and visualizes light intensity patterns
- **Horizon Detection**: Automatically locates horizon line using Hough transform
- **Cloudiness Estimation**: Estimates cloud coverage from image statistics
- **Astronomical Comparison**: Compares observed positions with theoretical solar ephemeris
- **Scientific Visualization**: Publication-ready graphs with dark theme and high DPI
